# The Travel Lab

Children's educational mission app by C442 Apps.
Built with React + Vite. Deploys to Vercel.

## What's inside
- Homepage (Adventures That Build Brilliant Minds)
- Mission Packs library (9 missions, Mission 001 live)
- Mission 001: Future Architect — full interactive pack (9 challenges, tap-to-explore panels, drawing canvases, XP tracker, ranks, certificate)
- Certificate generator
- Mission Control, About, Free Taster pages
- My Way accessibility panel

## Run locally (optional)
```
npm install
npm run dev
```

## Deploy
Connected to Vercel — pushes to the main branch deploy automatically.
